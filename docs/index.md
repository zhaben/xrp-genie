# 🧞‍♂️ XRP Genie Documentation

CLI tool for rapidly scaffolding XRPL wallet applications.

## 🚀 Quick Start

```bash
git clone https://github.com/zhaben/xrp-genie.git
cd xrp-genie
npm install
./cli/bin/xrp-genie init my-wallet-app
```
